# FWI Reconstruction using Implicit Neural Representation (INR)

##  Project Overview
This project explores full waveform inversion (FWI) using implicit neural representation (INR) for sound velocity reconstruction.

The project investigates neural-network-based reconstruction methods and compares different model configurations for image quality improvement.

---

##  Key Features
- Implemented INR-based reconstruction framework
- Explored SIREN and KAN-related network structures
- Compared different training configurations
- Evaluated reconstruction quality using PSNR, SSIM, and MRSE
